# 機能提案一覧

LLM API 料金比較ツールに追加すると良い機能をまとめたドキュメント。

---

## 1. データ拡充

### 1.1 プロバイダーの追加

- **Mistral AI** — Mistral Large, Medium, Small 等
- **Cohere** — Command R+, Command R 等
- **Amazon Bedrock** — Bedrock 経由の各モデル料金
- **Azure OpenAI** — Azure 独自の料金体系
- **Meta (Llama API)** — Llama 3.x シリーズ
- **xAI (Grok)** — Grok シリーズ

### 1.2 料金データの自動更新

- 各プロバイダーの公式料金ページをスクレイピングまたは API 取得して自動更新
- 更新日時の表示と差分ハイライト（値上げ/値下げの視覚化）
- 料金変更履歴の保持

### 1.3 モデル情報の拡充

- ベンチマークスコア（MMLU, HumanEval 等）の追加
- レート制限（RPM / TPM）情報
- 対応モダリティ（テキスト、画像、音声、動画）の明示
- SLA・稼働率情報

---

## 2. UI/UX 改善

### 2.1 ダークモード対応

- システム設定に連動したダークモード切り替え
- トグルボタンによる手動切り替え

### 2.2 モデル比較機能

- 2〜4 モデルを選択して横並び比較できるビュー
- 選択したモデルの料金差分をパーセンテージで表示
- 比較結果の URL シェア機能

### 2.3 グラフ・チャート表示

- 入力/出力料金の散布図（コンテキストウィンドウ vs 価格）
- プロバイダー別の価格帯分布（棒グラフ・箱ひげ図）
- 時系列での料金推移グラフ（値下げトレンドの可視化）

### 2.4 テーブル表示の改善

- 列の表示/非表示の切り替え
- 列の並び替え（ドラッグ＆ドロップ）
- ページネーションまたは仮想スクロール
- 行のピン留め（気になるモデルを上部に固定）

### 2.5 レスポンシブ対応の強化

- モバイルではカード表示に切り替え
- スワイプでの水平スクロール対応

---

## 3. 計算・分析機能

### 3.1 コストシミュレーター

- トークン数を入力して実際のコストを計算
- 月間利用量（リクエスト数 x 平均トークン数）から月額コストを試算
- キャッシュ利用率を考慮したコスト計算

### 3.2 コストパフォーマンス分析

- ベンチマークスコア / 価格 でコスパランキング
- ユースケース別のおすすめモデル提案（チャット、コード生成、要約等）
- 「予算内で最も高性能なモデル」のフィルタリング

### 3.3 通貨切り替え

- USD 以外の通貨での表示（JPY, EUR, GBP 等）
- リアルタイム為替レート連動

### 3.4 単位切り替え

- 1M トークンあたり / 1K トークンあたりの表示切り替え
- 文字数ベースでの概算表示（日本語・英語で異なるトークン効率を考慮）

---

## 4. ユーザー体験

### 4.1 お気に入り・ブックマーク

- よく参照するモデルをお気に入り登録
- ローカルストレージに保存して次回訪問時に復元

### 4.2 フィルタープリセット

- 「コスパ重視」「高性能重視」「低コスト」等のプリセットフィルター
- ユーザーが作成したカスタムフィルターの保存

### 4.3 URL パラメータ連動

- フィルター・ソート状態を URL クエリパラメータに反映
- 特定の表示状態を URL でシェア可能に

### 4.4 エクスポート機能

- 表示中のデータを CSV / JSON でダウンロード
- 比較結果の画像エクスポート（OGP 画像生成）

---

## 5. 技術基盤

### 5.1 API 化

- 料金データを REST API として公開
- 外部サービスやスクリプトからの利用を可能に

### 5.2 多言語対応（i18n）

- 日本語 / 英語の切り替え
- next-intl 等を使った本格的な国際化

### 5.3 PWA 対応

- オフラインでも閲覧可能に
- ホーム画面への追加対応

### 5.4 SEO 強化

- 各プロバイダー別のページ（`/openai`, `/google`, `/anthropic`）
- 構造化データ（JSON-LD）の追加
- OGP メタタグの最適化

### 5.5 パフォーマンス最適化

- 大量データの仮想スクロール
- データのインクリメンタルロード
- ISR（Incremental Static Regeneration）による静的生成

---

## 優先度の目安

| 優先度 | 機能                 | 理由                           |
| ------ | -------------------- | ------------------------------ |
| 高     | コストシミュレーター | ユーザーにとって最も実用的     |
| 高     | プロバイダー追加     | データの網羅性向上             |
| 高     | URL パラメータ連動   | シェアしやすさ                 |
| 中     | ダークモード         | 基本的な UX 改善               |
| 中     | グラフ表示           | 視覚的な比較が容易に           |
| 中     | モデル比較機能       | 意思決定を支援                 |
| 中     | 通貨・単位切り替え   | 国際ユーザー対応               |
| 低     | PWA 対応             | あると嬉しい機能               |
| 低     | API 化               | 開発者向け機能                 |
| 低     | 料金自動更新         | 運用負荷の軽減（実装コスト高） |
